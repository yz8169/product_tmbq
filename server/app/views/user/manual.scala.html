  • 使用说明

1. 简介

在靶向代谢组学研究中，大规模的代谢物定量仍然是个挑战。TMBQ（Targeted Metabolome Batch Quantification）是一个高通量的定制化的靶向代谢物定量平台。本平台包含6个主要模块：原始数据预处理（raw data preprocess）、峰识别（peak identification）、标准曲线拟合（standard curve fitting）、浓度计算（concentration calculation）、单位换算（unit conversion）和关键参数优化（key parameter optimization）。目前TMBQ仅支持以Waters或Agilent的UPLC/TQ-MS设备产生的靶向代谢物组数据。未来，更多供应商的数据格式也会得到支持。

2. 工作流程和主要特点


  • 完成原始LC / MS数据与浓度矩阵之间的连接；
  • 采用自主研发的动态递归拟合（Dynamic and Recursive Calibration）算法，全面考虑了样品浓度范围，拟合优度和最佳梯度点组合；
  • 峰识别和峰强度积分的参数相互独立，更加准确且真实；
  • 输入简单，只需要数据文件（zip.）和2个配置文件；
  • 结果与许多下游应用程序完全兼容；
  • 支持来自Agilent和Waters的UPLC-TQ / MS仪器的原始数据（需要通过Proteowizard进行格式转换）；
  • 可半自动化的确定关键参数；
  • 提供多种单位转化功能；
  • 高速的多线程运算（100多个样品，300多个代谢物只需要5分钟）；

3. 输入文件的准备

TMBQ需要导入以下文件：

  • 样本列表（.xls格式），
  • 保留时间列表（.xls格式），
  • 原始数据文件（.zip格式）。

3.1 样本列表

样本列表包含样本的批次信息（1,2,…），原始数据文件名，样本类型（标品或分析物）和标品浓度。至少需要1个浓度梯度（如Conc A），也允许一个以上的浓度梯度（最多50个）。每一个批次至少需要2个标准样品和1个分析样品。

3.2 保留时间列表

保留时间列表包含待定量的代谢物编号以及保留时间。
